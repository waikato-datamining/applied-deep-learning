The following prerequisites apply to most data domains:

Annotations

In order to annotate data (e.g., for object detection, instance segmentation or image segmentation), you should download the ADAMS Annotator application (you only need Java 11 installed):

  • Snapshot
  • Release 2021.12.0 or later

Format conversions

For turning annotations from one format into another, you need to install the wai.annotations Python library:

  • wai.annotations project
  • wai.annotations manual

NB: it is recommended to install it in a virtual environment to avoid any conflicts with the host system.

Hardware

For building models, a computer with NVIDIA GPU (8GB+) and Linux operating system is recommended. Only image classification models can be built in a reasonable amount of time on CPU-only machines.

Directory structure

Create the following directory structure for the examples of this tutorial:

|
+-- applied_deep_learning
    |
    +-- data
    |
    +-- output
    |
    +-- predictions
        |
        +-- in
        |
        +-- out
